# @baseui-cn/www

Documentation site for baseui-cn.

## Local development

```bash
pnpm --filter @baseui-cn/www dev
```

Open `http://localhost:3000`.

## Generated assets

- `public/registry/*` is generated by `node scripts/build-registry.mjs`
- `lib/__generated__/catalog.json` is generated by `node scripts/build-registry.mjs`
- `lib/__generated__/preview-sources.json` is generated by `node scripts/build-preview-sources.mjs`

Run the generators from the repo root after changing installable metadata, UI sources, or previews.
