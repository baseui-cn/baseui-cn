## API

### Toggle

| Prop      | Type                        | Default     | Description                                                         |
| --------- | --------------------------- | ----------- | ------------------------------------------------------------------- |
| `variant` | `"default" \| "outline"`    | `"default"` | Border and background treatment — transparent (default) or outlined |
| `size`    | `"sm" \| "default" \| "lg"` | `"default"` | Height, min-width, and padding scale                                |
