## API

### Input

| Prop          | Type                                | Default | Description                                                        |
| ------------- | ----------------------------------- | ------- | ------------------------------------------------------------------ |
| `size`        | `"sm" \| "default" \| "md" \| "lg"` | `"md"`  | Height and padding scale of the input                              |
| `unstyled`    | `boolean`                           | `false` | Strip the outer wrapper styles for composition inside `InputGroup` |
| `nativeInput` | `boolean`                           | `false` | Render a plain `<input>` element instead of the Base UI primitive  |
